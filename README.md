# The Run – Blockchain Based Asset Trading Software

## Overview

**The Run** is a blockchain-based asset trading software designed to provide a secure, transparent, and decentralized platform for trading digital assets. The platform enables users to buy, sell, and exchange virtual assets such as cars, skins, collectibles, and NFTs using blockchain technology.

The project leverages Ethereum smart contracts to ensure ownership verification, secure transactions, and tamper-proof asset management.

## Features

- Secure buying, selling, and trading of digital assets
- Smart contract-based automated transactions
- NFT-powered ownership verification
- Transparent blockchain records
- Ethereum-powered ecosystem

## Tech Stack

- Ethereum
- Solidity v0.8.20
- Hardhat
- OpenZeppelin Contracts
- Ethers.js

## Installation

```bash
npm install
npx hardhat compile
npx hardhat node
npx hardhat ignition deploy ./ignition/modules/Lock.js --network localhost
```
